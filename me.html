Портфолио

  • Creditbook (2011-2014 гг)
    Разработка компонентов рейтинга банков, каталога кредитов, кредитных калькуляторов, а так же практически всех остальных компонентов, модулей и плагинов, представленных на сайте. Верстка сайта.
  • Чай вдвоём (2014-2016 гг)
    Разработка компонента блогов с функциями социальной сети. Компонент содержит: блоги, профили пользователей, систему комментариев, обмен личными сообщениями, чат, подписки (на весь блог, отдельные материалы, отдельных блоггеров), возможность добавлять в друзья, всплывающие уведомления наподобии "Вконтакте". Записи блогов группируются на основе мультикатегорий и тегов. Компонент является развитием ранее написанного мной "Hekima Blog". На сайте так же используется модификация плагина Zygo Profile
  • CasualWall - магазин молодежной одежды. (2017 г)
    • Создание магазина (Joomshopping)
    • Разработка компонента Zygo_Jshopping_Frontend для добавления и редактирования товаров, аттрибутов и характеристик товаров с фронт-энда.
    • Написание модуля категорий и производителей (в выпадающем меню "Ассортимент").
    • Создание плагинов: Zygo Joomshopping Review Captcha - вывод капчи KCaptcha для отзывов в интернет-магазине Joomshopping. NorrTrickyModules - плагин вывода модулей по определенным логическим условиям GET и POST-запросов. С помощью этого плагина выводятся такие модули как: боковой модуль "Новые поступления" - выводится только при заходе в товар; модуль распродажи - "Все до 3000", который выводится, если в магазине в поиске устанавливается цена меньше 3000) и ряд других модулей
  • Написание плагинов для социальной сети JomSocial под cms Joomla, в частности, js announcer (2013)
  • ПузиРадуга - написание компонента организации конкурсов (2012 г)
  • Разработка каталога автомобилей, подбора автомобиля по комплексу параметров и получения выгодного автокредита при покупке автомобиля. (сайт изменил владельца) (2012 г)
  • Ruscargo - сайт-визитка холдинга компаний-грузоперевозчиков (2011 г)
  • CherryEstates - создание сайта и интранет базы данных агенства недвижимости (2010 г)
  • Camino Latino - создание сайта культурного центра Латинской Америки и перевод зарубежного контента для сайта (2009 г)
  • Занимательная педагогика- создание и сопровождение сайта, написание компонента аякс-тестов с использованием функционала jquery-ui (2008 г)
